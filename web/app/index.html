atlas

Runs entirely in your browser. No sign-in.

See any GitHub repo as a living map.

Atlas reads a project's specs, source, and git history and draws one interactive picture: a spec and code graph, a coverage treemap and sunburst, a dependency DAG, a churn-vs-coverage view, and more. The whole engine is compiled to WebAssembly and runs right here on the page.

Works on any public repository. Repos with no specs still get a treemap, language mix, and orphan clusters.

Private repo, or hitting rate limits? Add a token (optional)

Anonymous requests share GitHub's 60-per-hour limit. Paste a personal access token (classic, with repo scope for private repos, or no scope for public) to get 5,000 per hour. It is stored only in this browser and sent only to api.github.com.
